About Me

Ever since finding that I could program formulas into a graphing calculator, I became interested in using programming to make life even slightly more efficient. I graduated with a Bachelor of Science in Computer Science, but decided that I wanted more practical experience, so after working as a math tutor, I decided to take the leap and enroll in Fullstack Academy to further cultivate my technical and collaborative skills and grow as both a programmer and a person. I have fullstack Javascript experience with a focus on the NERD stack (Node.js, Express, React/Redux, and PostgreSQL (Sequelize)), as well as experience with Java (and a few other technologies) from my projects while at Rutgers. But feel free to check out my GitHub to see everything that I've worked on, and don't hesitate to reach out to grab a coffee sometime!

Projects

Timey is a NERD web application that will allow users to keep track of the time they spend on each task. Users are provided a timesheet detailing all of the time spent (filtered by ticket, project, start date, end date) as well as graphs portraying the data in a more digestible manner. Users can drag and drop between task columns, create, edit, and remove tickets in their projects, all of which is immediately updated for any other user also watching the project.
